A Tag Target in HTML



Syntax <a target="_blank|_self|_parent|_top|framename">
  1. _blank :- Opens the linked document in a new window or tab Example :- Google
  2. _self :- Opens the linked document in the same frame as it was clicked (this is default) Example :- Google

  3. _parent :- Opens the linked document in the parent frame

  4. _top :- Opens the linked document in the full body of the window

  5. framename :- Opens the linked document in the named iframe
